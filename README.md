# fast-png

  [![NPM version][npm-image]][npm-url]
  [![npm download][download-image]][download-url]

PNG image decoder and encoder written entirely in JavaScript

## Installation

```
$ npm install --save fast-png
```

## Usage

### `decode(png)`

### `encode(png, options)`

## PNG standard

Spec can be found at: https://www.w3.org/TR/PNG/
